# **App Name**: EGS Admissions Hub

## Core Features:

- Admission Form: A comprehensive and user-friendly admission form with fields for Name, Email, Phone, Program Selection, and State.
- Program Categories Display: Interactive grid to display program categories like Engineering, Arts & Science, and more.
- Detailed Program Highlights: Dynamic listing of programs such as BE, B.Tech, B.Sc with 'Know More' CTAs for in-depth information.
- Testimonial Slider: Rotating slider showcasing student and faculty testimonials to build trust and credibility.
- Placement Partner Showcase: Logo grid of top recruiters to highlight placement opportunities and industry connections.
- SEO and Ads Optimization: Implementation of SEO best practices and ad-friendly URL structures with UTM parameters for tracking and deep linking.

## Style Guidelines:

- Primary color: Dark navy blue (#00176E) to convey trust, stability, and academic strength.
- Background color: Very light desaturated blue (#EAF0F8). A light background will optimize readability and performance.
- Accent color: Fuchsia (#AF005E) to add highlights and emphasize important sections; reinforces the institution's brand identity.
- Headline font: 'Poppins' sans-serif for high-contrast headlines. Its geometric style provides a contemporary and fashionable look, well-suited for impactful headlines and short amounts of text.
- Body font: 'PT Sans' sans-serif; to complement Poppins and create a more relaxed and pleasant reading experience.
- Use flat design icons to represent program categories, admission steps, and campus life elements.
- Maintain a structured and clean layout with clear headings, soft shadows, and rounded form fields.
